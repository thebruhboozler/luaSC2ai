new (name, race, ip, port)
creates sc2ai agent object

Parameters:

  • name string name of the bot
  • race nmber race of the bot (1 for terran , 2 for zeg , 3 for protos and 4 for random)
  • ip ?string|nil ip where starcraft 2 expects the connection , if nil then it will use the environment varriable SC2IP
  • port ?string|nil port where starcraft 2 expects the connection , if nil then it will use the environment varriable SC2PORT

Returns:

    table agent
createGame (mapPath, enemyRace, autoJoin, disableFog, randomSeed, realtime)
tells starcraft 2 to create the match

Parameters:

  • mapPath string path to the .SC2MAP file
  • enemyRace number race of the opponent ( 1 Terran , 2 Zerg , 3 Protos , 4 random)
  • autoJoin ?boolean|nil will automatically join the game upon the match being created , by default is equal to true
  • disableFog ?boolean|nil will disable fog , by default is equal to true
  • randomSeed ?number|nil the seed which starcraft 2 will use , if not passed starcraft 2 will use its own seed
  • realtime ?boolean|nil if the game should run in realtime or not (22.4 updates per second)

findNearestUnitOfType (target, unitType, alliance, targetAlliance)
finds the neartest unit of type from a point or an unit

Parameters:

  • target ?table|number the coordinates { x , y } or a tag of an unit , if passed in as a tag it will use the units coordinatesand find the neartest to it
  • unitType number which type of an unit to find
  • alliance string the alliance of the unit being searched for "Self" for self, "Enemy" for enemies , "Ally" for allies and "Neutral" for neutral units (meaing mineral fields or barries on the map)
  • targetAlliance ?string|nil the alliance of the unit we are searching from , if the target is given as a table of coordinates then this parameter is ignored

Returns:

    ?table|nil the nearest unit found or nil if no unit found
